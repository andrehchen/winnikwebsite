# Reference: "Minimal" portfolio template (via Land-book search)

Found via land-book.com, searching "minimal". Note: this is a paid Framer
template listing (not a live independent designer's site) — the structural
pattern is still a strong match for Winnik's hairline/editorial aesthetic,
so borrow the layout logic, not any of the template's own sample copy or
placeholder project art.

## What to borrow

- **Nav as a plain text row with a hairline rule underneath**: wordmark at
  far left, 2-3 plain text links right-aligned, a single thin horizontal
  line spanning the full width sits directly below the row (not a shadow,
  not a background band — just a 1px hairline). This is a lighter-weight
  nav treatment than a bordered header bar and would suit the Winnik site's
  already-minimal `.site-header`.
- **Single oversized editorial statement as the entire hero**: no image,
  no icon — just 3-4 short sentences set large (roughly 32-40px), left
  aligned, ragged right edge, generous line-height, sitting in a mostly
  empty page with large margins. Reads as confident/quiet rather than
  "unfinished." Could work well as an alternate treatment for a sub-page
  intro (e.g. Winnik's own bio intro) if the page ever feels like it needs
  less visual machinery than the homepage's PEG animation.
- **Two-column masonry-style project grid below the statement**: each card
  is a plain rectangle with a flat color fill and a single line of title +
  subtitle beneath it (title bold, subtitle muted, small caps-ish label at
  bottom-right like a category tag). Cards are not uniform height — some
  taller, some shorter — giving a loose masonry rhythm rather than a rigid
  grid. For Winnik this maps onto something like the People grid or a
  Publications-by-year layout: uniform card *treatment* (image/fill +
  title + subtitle + tag) but non-uniform card *height* reads less
  corporate than a strict grid.

## What NOT to carry over

- The specific color-blocked card fills (indigo/black/pale-yellow/blush) —
  Winnik is strict monochrome; translate "flat color card" into "flat
  paper/band-gray card with a hairline border" instead.
- Any of the template's own placeholder copy, names, or project thumbnails
  — these are that template's sample content, not something to reference
  or adapt text from.
