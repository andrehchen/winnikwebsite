# Winnik Group Website

A rebuild of the Winnik Research Group's lab website. The old site
(`winnikgroup.chem.utoronto.ca`, built on Grav CMS) was outdated, so this
replaces it with a more interactive, modern site — same content, same
identity, dramatically better execution.

Static HTML/CSS/JS, no build step. See `BRIEF.md` for the full design brief
and content inventory, and `CLAUDE.md` for the build rules.

Run locally with:

```
python3 -m http.server 8000
```
